## 🐻 Блитц (Blitz)
**Блитц (Blitz)** — интернет-магазин по продаже корма для собак и кошек.
!<img width="481" height="174" alt="image" src="https://github.com/user-attachments/assets/4abd3858-251b-4433-b759-187f23388e03" />


##  Возможности

### 1. Интерактивный квиз (Умный подбор рациона)
Пошаговая ветеринарная анкета для владельцев, симулирующая подбор идеального питания:
* **Сбор параметров:** вид животного (кошка/собака), возраст, вес и тд

### 2. Архитектура каталога и фильтрация
Реализована гибкая реляционная структура БД для точного поиска:
* **Обширная фильтрация:** сквозной поиск классу корма (супер-премиум), назначению, возрасту, порода и тд

### 3. Динамическая корзина
Удобный и отзывчивый интерфейс управления покупками без перезагрузки страниц:
* **Интерактивность:** мгновенное добавление и удаление товаров 
* **Синхронизация:** автоматический пересчет итоговой стоимости заказа

 ## Технологический стек
Backend: Python 3.13, Django 6.0
Database: PostgreSQL (перенос с SQLite)
Frontend: HTML5, CSS3, Vanilla JavaScript, элементы Bootstrap
Testing: Django TestCase (Unit-тестирование корзины, авторизации, лимитов склада)

__________________________________________________________________________________________________________________________________________

 ## Установка и запуск 
Установка

Установить Django:
pip install django==5

Перейдите в cd food

Запустите сервер разработки:
python manage.py runserver
